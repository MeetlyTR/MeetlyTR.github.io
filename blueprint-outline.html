Installation

Add BlueprintOutline to your Unreal Engine project

1. Download the Plugin

Download the BlueprintOutline plugin from GitHub or Unreal Engine Marketplace.

2. Copy to Plugin Folder

Copy the downloaded plugin to your Unreal Engine project's Plugins folder.

YourProject/ ‚îî‚îÄ‚îÄ Plugins/ ‚îî‚îÄ‚îÄ BlueprintOutline/

3. Enable the Plugin

In Unreal Engine Editor, find the BlueprintOutline plugin in Edit > Plugins menu and enable it. Restart the Editor.

4. Start Using

Open Blueprint Editor and start using BlueprintOutline features. Outline visualization will be automatically active.
